# Soren Store Frontend

React + TypeScript storefront for the Soren portfolio e-commerce app.

## Stack
- React + TypeScript
- React Router
- MUI
- Apollo Client

## Run
```bash
yarn install
yarn start
```

If backend URL differs from `http://localhost:3000/graphql`, set:
```env
REACT_APP_GRAPHQL_URL=...
```

## Verify
```bash
yarn test --watchAll=false
yarn build
```
